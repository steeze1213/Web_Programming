# Web Programming

학교 수업 웹프로그래밍을 한 학기 동안 수강하며 실습, 과제로 작성한 코드 모음입니다.  
HTML, CSS, JavaScript의 기초 문법과 웹 페이지 동작 원리를 학습하는 것을 목표로 한 개인 학습 기록용 레포지토리입니다.

---

## 사용 교재

HTML5 + CSS3 + JavaScript로 배우는 웹프로그래밍 기초
- 출판 연도: 2019  

---

## 학습 내용 요약

### HTML
- HTML 기본 요소
- HTML 멀티미디어와 기본 양식

### CSS
- CSS3 스타일 시트 기초
- CSS3 박스 모델과 응용
- CSS3 레이아웃과 애니메이션
- CSS3 HTML과 CSS로 웹 사이트 만들기

### JavaScript
- 자바스크립트 기초
- 자바스크립트 객체
- DOM과 이벤트 처리, 입력 검증
- 자바스크립트와 캔버스로 게임 만들기

### 고급 기술
- jQUERY, Ajax, JSON
- HTML 위치 정보와 드래그와 드롭
- HTML5 웹 스토리지, 파일 API, 웹 소켓
- JSP
- 모바일 웹 페이지
